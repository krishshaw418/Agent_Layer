## Get Started

Follow this flow to create an API key, fund the vault, and prepare your account for SDK usage.

## Step 1: Connect wallet and sign up

Open the app, connect your wallet, and complete signup.

## Step 2: Open API Keys

Click the **Get API key** button, then go to the [API Keys page](/api-keys).

## Step 3: Create a new API key

On the API Keys page, click **Create New API Key** and generate a key for your app, service, or node runner.

## Step 4: Purchase AGL tokens

If you are a developer using the Agent Layer SDK, click **Purchase AGL Token** and buy AGL tokens using USDC.

## Step 5: Recharge the vault

Return to the [API Keys page](/api-keys) and click **Recharge Vault**. This top-up is used for LLM token purchases so you can use the SDK.

<Callout>
  The API key generation flow is the same for node runners and developers using the Agent Layer SDK.
</Callout>

<Callout>
  Node runners do not need to purchase AGL tokens. They receive AGL tokens as incentives, which they can check on the /check-agl-balance page.
</Callout>

## Quick summary

1. Connect wallet and sign up.
2. Click **Get API key**.
3. Click **Create New API Key**.
4. Developers: purchase AGL tokens using USDC.
5. Return to **API Keys** and click **Recharge Vault**.
