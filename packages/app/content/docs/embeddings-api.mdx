## Create embeddings

Use the embeddings resource when you need vector representations for search, retrieval, ranking, or clustering.

<ExampleTabs
  curl={String.raw`curl https://api.agentlayer.ai/embeddings \
  -H "Authorization: Bearer $AGENT_LAYER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "agent-embed-1",
    "input": "Decentralized AI infrastructure"
  }'`}
  javascript={String.raw`const embedding = await client.embeddings.create({
  model: "agent-embed-1",
  input: "Decentralized AI infrastructure"
});

console.log(embedding.data[0].embedding.length);`}
  typescript={String.raw`const embedding = await client.embeddings.create({
  model: "agent-embed-1",
  input: "Decentralized AI infrastructure"
});

console.log(embedding.data[0].embedding.length);`}
/>

## Request fields

- `model`: embedding model identifier.
- `input`: one string or an array of strings.
